# **App Name**: Strength Up Pro

## Core Features:

- Dynamic Content Showcase: Present all essential gym information including Hero, About, Membership Plans, Services, Trainers, Gallery, and Reviews sections in a clean and organized manner.
- Membership Plan Highlight: Visually showcase different membership tiers (Basic, Standard, Premium), highlighting the best value option and including clear calls-to-action for each plan.
- Direct Communication Hub: Integrate interactive elements for immediate communication, such as clickable WhatsApp links, direct phone dialer buttons, and an embedded Google Maps location for easy navigation.
- Responsive & Performance Optimization: Ensure the website is fully responsive, optimized for mobile-first browsing, features a sticky navigation bar for ease of access, provides smooth scrolling, and ensures fast loading times for a premium user experience.
- Fitness Goal Assistant: An interactive tool powered by AI that, based on a user's selected primary fitness goal (e.g., build strength, improve cardio, lose weight), generates a concise and generalized piece of advice or a basic, example exercise recommendation. This provides immediate, personalized (albeit simple) value to site visitors.

## Style Guidelines:

- Primary color: A vibrant, bold orange (#FFAC03) for an energetic and premium accent, used for CTAs and key highlights. Its hue reflects ambition and vitality.
- Background color: A very dark, subtle charcoal (#211D15), derived from the primary color's hue but heavily desaturated, providing a deep and luxurious canvas for the dark theme.
- Accent color: A rich, burnt orange-red (#CF4817), analogous to the primary but with significantly different brightness and saturation to offer a powerful, complementary highlight.
- Headlines and body text font: 'Inter' (sans-serif) for a modern, clean, and objective look that ensures high readability and a professional aesthetic throughout the site.
- Utilize sleek, minimalist vector icons that visually represent fitness, strength, and health, reinforcing the clean and premium brand image without clutter.
- Adopt a mobile-first, responsive grid-based layout. Content sections should be clean, structured, and incorporate visual breaks. Implement a sticky navigation bar for consistent access and smooth scrolling between page sections.
- Integrate subtle, tasteful scroll-triggered animations (e.g., gentle fade-ins or slight parallax effects) on different sections to add depth and elegance, enhancing user engagement without distracting from the core content.
